<p>안녕하세요. 제이콥입니다.</p>
<p>Wordpress 글을 Steem으로 쉽게 옮길 수 있는</p>
<p>워드프레스 플러그인을 소개합니다.</p>

<hr />
<p> </p>
<h2>워프스팀 – 플러그인 소개</h2>
<ul>
<li>워드프레스 하나로 블록체인에 글을 써보자.</li>
<li>워프스팀은 여러 자원(인스타, 페이스북, 블로그, 스팀 등등)에서 쉽게 스팀으로 <strong>‘워프’</strong>, 이동할 수 있도록 합니다.
<ul>
<li>사용자는 자신의 글을 <strong>홍보</strong>하거나 <strong>부수입</strong> 창출할 수 있습니다.</li>
<li><strong>이전에 작성한 훌륭한 글</strong>들을 스팀으로 옮길 수 있습니다.</li>
</ul>
</li>
<li>첫 시작으로는 <strong>wordpress</strong>에서 steem으로 글을 쉽게 이동하게 합니다.
<ul>
<li>현재 플러그인을 배포하였으며, 다운로드 가능합니다.</li>
<li><strong>기존에 있던 한국 워드프레스 사용자</strong>들이 쉽게 Steemit에 접근할 수 있도록 합니다.</li>
</ul>
</li>
</ul>
<hr />
<h2>워프스팀</h2>
<p>관리자페이지 -> 플러그인 -> 새로 추가에서 WarpSteem을 검색하면 다운받을 수 있습니다.</p>
<p><a href="https://ko.wordpress.org/plugins/warpsteem/">https://ko.wordpress.org/plugins/warpsteem/</a></p>
<p><img class="alignnone wp-image-1051 " src="https://passionbull.net/wp-content/uploads/2018/10/warpsteem.png" alt="" width="501" height="230" srcset="![](https://passionbull.net/wp-content/uploads/2018/10/warpsteem.png) 562w, ![](https://passionbull.net/wp-content/uploads/2018/10/warpsteem-300x138.png) 300w" sizes="(max-width: 501px) 100vw, 501px" /></p>
<hr />
<h2>사용 방법 – 설정</h2>
<p>1. 관리자 페이지 -> 설정 -> WarpSteem setting으로 들어갑니다.</p>
<p>2. 순서대로 스팀잇 아이디, <strong>Posting key</strong>, 기본 tag를 적습니다.</p>
<p>3. ‘Enable’ 체크박스를 체크합니다.</p>
<p>워드프레스 새 글을 작성하거나 수정하면, 자동으로 스팀잇에도 업데이트 됩니다.</p>
<p><img class="alignnone wp-image-1052 " src="https://passionbull.net/wp-content/uploads/2018/10/screenshot-1.png" alt="" width="499" height="383" srcset="![](https://passionbull.net/wp-content/uploads/2018/10/screenshot-1.png) 1685w, ![](https://passionbull.net/wp-content/uploads/2018/10/screenshot-1-300x230.png) 300w, ![](https://passionbull.net/wp-content/uploads/2018/10/screenshot-1-768x589.png) 768w, ![](https://passionbull.net/wp-content/uploads/2018/10/screenshot-1-1024x786.png) 1024w" sizes="(max-width: 499px) 100vw, 499px" /></p>
<hr />
<h2>사용 방법 – 글 작성</h2>
<p>워드프레스를 이용하여 평소대로 글을 작성합니다.</p>
<p>그리고 공개 버튼을 누릅니다. 끝!</p>
<hr />
<p>사용자를 위한 추가 설정이 몇 개 있습니다.</p>
<p>옆에 포스트 박스를 통해서 태그를 설정할 수 있습니다.</p>
<p>설정된 태그는 스팀잇에 적용됩니다. (인스타그램의 태그)</p>
<p>고유주소에서 permlink를 수정하면 그대로 스팀에도 적용됩니다. (한글로 설정할 경우, post id로 설정됩니다.)</p>
<p>또한, 고유주소도 자신이 원하는대로 설정할 수 있습니다.</p>
<p>하지만 같은 글에 대해 고유주소를 변경하면 중복된 글이 작성됩니다.</p>
<p><img class="alignnone size-full" src="https://ipfs.busy.org/ipfs/QmSpD9v1ahoB2C8EUksSw4bnNH4j5dvZATT9djGkmMfvwL" width="960" height="451" /></p>
<hr />
<h2>계획</h2>
<ul>
<li>유입된 워드프레스 유저들의 글 홍보 및 보팅</li>
<li>Steem 계정 생성 지원
<ul>
<li><strong>워드프레스 유저들의 계정을 생성해주고 싶습니다.</strong></li>
</ul>
</li>
<li>개발 관련
<ul>
<li>Editor 내에서 update on/off</li>
<li>불편한 사항 접수하여 보완</li>
</ul>
</li>
</ul>
<hr />
<h2><strong>참고사항</strong></h2>
<p>Steemit에서는 여러 Key가 있습니다.</p>
<p><strong>Posting key</strong>는 스팀잇에 글을 쓰기 위해 필요한 key입니다.</p>
<p>이 플러그인은 Postking key (only 글쓰기, 보팅만 가능) 가 필요합니다.</p>
<p><a href="https://steemit.com/@유저_아이디/permissions">https://steemit.com/@유저_아이디/permissions</a>에 접속하여 Posting key를 복사하여</p>
<p>워드프레스 설정에 넣습니다.</p>
<p><img class="alignnone wp-image-1053" src="https://passionbull.net/wp-content/uploads/2018/10/screenshot-2.png" alt="" width="701" height="274" srcset="![](https://passionbull.net/wp-content/uploads/2018/10/screenshot-2.png) 2497w, ![](https://passionbull.net/wp-content/uploads/2018/10/screenshot-2-300x117.png) 300w, ![](https://passionbull.net/wp-content/uploads/2018/10/screenshot-2-768x300.png) 768w, ![](https://passionbull.net/wp-content/uploads/2018/10/screenshot-2-1024x400.png) 1024w" sizes="(max-width: 701px) 100vw, 701px" /></p>
<p>이 플러그인을 활용하여 글을 작성할 경우, Warpsteem이 글 작성 수익의 5%를 가져갑니다.</p>
<p>새로 오시는 워드프레스 유저님들을 지원하는데 사용하겠습니다.</p>
<ul>
<li>유입된 워드프레스 유저들의 글 홍보 및 보팅</li>
<li>계정 생성 – 추후 가능</li>
</ul>

<li>보팅을 선별적으로 해준다.</li>
</ul>
</li>
</ul>
